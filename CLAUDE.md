# CLAUDE.md - 2025 Year in Review

3D constellation visualization showcasing what was built in 2025 with AI assistance.

**Live Site**: https://vishalsachdev.github.io/2025-in-review/
**Repo**: https://github.com/vishalsachdev/2025-in-review

---

## Tech Stack

- Three.js for 3D WebGL rendering
- Vanilla JavaScript (no framework)
- GitHub Pages deployment
- OrbitControls for camera navigation
- Raycaster for click detection
